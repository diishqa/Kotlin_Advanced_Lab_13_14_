## Сегодняшнеее изучение
**T — это плейсхолдер (placeholder) для типа данных;**
пример:
```
data class Question<T>(
    val questionText: String,
    val answer: T,
    val difficultly: Difficulty
)
```
___
**Extension-функции** объявляются аналогично —с указаниемтипа иточкипередименем функции

**Scope-функции** позволяют работать со свойствами и методамиобъекта,неповторяя имя переменной, 
если это избыточно. Это особенно часто используетсявAndroid-разработке

**Массив (Array)** — это простейший способ хранить несколько значенийодноготипа данных. 
- массив 
состоит из элементов (items);
- элементы упорядочены;
- доступ к элементам осуществляется по индексу
___
**MutableList** — расширяет List и добавляет методы для изменениясписка(add,remove и т.д.)

**Set** — это коллекция без фиксированного порядка, которая недопускаетдубликатов.
- элементы уникальны;
- порядок элементов не гарантирован;
- нет индексов

**Map** — это коллекция, состоящая из ключей и значений. Она называется «картой», потому что 
каждому уникальномуключусопоставляется значение.

___
- Arrays хранят упорядоченные данные того же типа и имеют фиксированныйразмер. 
- Arrays 
используются для реализации многих других типов коллекций. 
- Lists - это изменяемая, упорядоченная коллекция. 
- Sets являются неупорядоченными коллекциями и не могутсодержатьдубликатов. 
- Maps работают аналогично наборам и хранящимся парамключейизначенийуказанного типа
